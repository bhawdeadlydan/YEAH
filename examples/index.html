How to use:

Yeah is a jquery plugin! So it need jquery to works.
It's IMPORTANT! If you use bootstrap or any css framework that use float elements bind the load of plugin on window load event. 



$(window).load(function(){
    $(".animated").yeah();
});

Yeah is completely responsive! You can set availables width and mobile check! 


$(window).load(function(){
    $(".animated").yeah({
        responsiveAvailableWidth:[{min:768,max:1024},{min:1440,max:9999}],
        mobile:true
        });
});

Yeah is currently a beta version. Soon we will add others features likes public methods and events.
